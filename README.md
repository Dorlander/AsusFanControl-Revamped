# Asus Fan Control Revamped

### Запуск
Командная строка, старт: `AsusFanControl.exe <Процент скорости кулеров>`  
Командная строка, Выключить тест-мод: `AsusFanControl.exe 0`  
GUI: `AsusFanControlGUI.exe`
